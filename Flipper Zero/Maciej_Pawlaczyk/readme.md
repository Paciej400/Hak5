# Flipper Zero Demonstration

## Disclaimer

> [!WARNING]
> Project has been done for educational purposes only. Its aim is to demonstrate possibilities, limitations and threats of data transmitted via open ether (infrared light and radio waves in this case).<br/>
<br/>
All the experiements shown in this folder were done on authors' own devices.<br/>
<br/>
Respect privacy of others.

---
### Project overview

Flipper Zero is a device developed by Flipper Devices Inc. It runs on STM32WB55 microcontroller and has a few antennas and signal receivers/transmitters, which allows it to capture and replay a signal.

Project was developed in order to present it in front of other members of science club of the Gdańsk University of technology. In order to present the possibilities of the device, a film was made (available only in the Polish language). Discussion of the theoretical background (including mechanisms immune to replay attacks) was shown in the presentation in Canva (also in the Polish language).

---
### Aim of the project
- Present the general format of the data transmitted from simple devices
- Show the difference between fixed code and rolling code
- Raise awareness about existance of devices, which are able to capture signal
- Raise awareness for the need to care about security of the data transmitted via open ether, especially for critical sytems
